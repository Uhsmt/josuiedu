# 浄水システムアニメーション計画書

## プロジェクト概要
- **目的**: 子供向け学習用浄水システム説明アニメーション
- **対象**: 大画面モニターでの展示用
- **形式**: ループ再生可能な2Dアニメーション
- **納期**: 2週間で3本完成
- **技術**: JavaScript + HTML + SVG/CSS

## 制作予定の3本のアニメーション

### 1. 初沈池（しょちんち）
- **内容**: 大きなゴミや砂を沈殿させる工程
- **キーポイント**: 重力による自然沈降
- **アニメーション要素**: 
  - 汚れた水の流入
  - 重い粒子の沈降
  - 清澄な上澄み水の流出

### 2. 反応槽（生物処理）
- **内容**: 微生物による有機物分解
- **キーポイント**: 微生物の働き、酸素供給
- **アニメーション要素**:
  - 微生物キャラクター（すいすい君、クマムシマン等）
  - 泡の発生（酸素供給）
  - 有機物の分解過程

### 3. 終沈池（しゅうちんち）
- **内容**: 活性汚泥の分離・回収
- **キーポイント**: 掻き寄せ機の動作
- **アニメーション要素**:
  - 活性汚泥の沈降
  - 掻き寄せ機の回転
  - 清澄水の越流

## 技術仕様

### 基本構成
- **HTML5**: 構造とキャプション表示
- **SVG**: ベクター図形（配管、設備）
- **CSS Animation**: 水の流れ、機械の動作
- **JavaScript**: インタラクション制御

### 画面仕様
- **解像度**: レスポンシブ対応（4K対応）
- **縦横比**: 16:9推奨
- **キャプション**: 日本語、読みやすいフォントサイズ
- **ループ**: 無限ループ再生

### パフォーマンス要件
- **軽量化**: GPU加速利用
- **互換性**: モダンブラウザ対応
- **オフライン**: インターネット接続不要

## 既存素材の活用
- キャラクター: すいすい君（各方向）、クマムシマン、ミジンコ
- 参考資料: 工程写真、漫画形式の説明資料

## スケジュール
- **Week 1**: 3本すべての基本版を完成
- **Week 2**: 詳細調整、3D要素検討、最終調整

## 配布・展示方法
- **GitHub Pages**: 進捗共有・プレビュー
- **オフライン配布**: HTMLファイル一式
- **大画面対応**: フルスクリーン表示
